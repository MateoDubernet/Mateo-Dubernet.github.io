Application One Piece (Angular + Spring Boot)

Une application réaliser au début de mon alternance pour me former aux frameworks Angular et Spring utiliser dans l'entreprise. Cette application permet de gérer une liste de personnages de l'univers One Piece, avec des fonctionnalités complètes de création, lecture, mise à jour et suppression (CRUD). Inclut également une fonctionnalités de déplacement d'une image sur une grille via des boutons de direction. Elle utilise Angular pour le frontend et Spring Boot pour le backend, avec une base de données PostgreSQL pour le stockage des données.
